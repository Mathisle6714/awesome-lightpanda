2. Look for the latest release near the top of the page.

4. Find the file made for Windows. It may end in `.exe`, `.zip`, or `.msi`.
5. If you get a `.zip` file, right-click it and choose **Extract All**.

8. If Windows shows a security prompt, choose **More info** and then **Run anyway** if you trust the source.
